here we use some html tag to build our personal details form:-
1) div tag:- The <div> tag defines a division or a section in an HTML document.
2) select tag:-The <select> element is used to create a drop-down list.
The <select> element is most often used in a form, to collect user input.
3) label tag:- The <label> tag defines a label for several elements:

<input type="checkbox">
<input type="color">
<input type="date">
<input type="datetime-local">
<input type="email">
<input type="file">
<input type="month">
<input type="number">
<input type="password">
<input type="radio">
<input type="range">
<input type="search">
<input type="tel">
<input type="text">
<input type="time"> etc..

4) type attribute :- For <button> elements, the type attribute specifies the type of button.
   For <input> elements, the type attribute specifies the type of <input> element to display.

5) placeholder attribute:- The placeholder attribute specifies a short hint that describes the expected value of an input field (e.g. a sample value or a short description of the expected format).
